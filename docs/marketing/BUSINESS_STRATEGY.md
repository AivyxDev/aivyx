# Business Strategy

> Aivyx market positioning, revenue model, and growth plan.

## Vision

**Own Your Intelligence** — Aivyx is the privacy-first AI agent for professionals who refuse to trade their data for convenience.

## Market Position

### The Gap

| Existing Solutions | Problem |
|-------------------|---------|
| ChatGPT, Claude, Gemini | Cloud-only, your data on their servers |
| Open-source LLM UIs | No memory, no persistence, no agent capabilities |
| Enterprise AI platforms | Complex, expensive, team-focused |

### Aivyx's Position

**Personal AI infrastructure** — runs locally, remembers everything, encrypts by default. For individual professionals who want AI that works *for* them, not *on* them.

## Target Users

### Primary: Technical Professionals

- **Developers** — code review, debugging, project management
- **Researchers** — literature synthesis, knowledge management
- **DevOps/SREs** — incident response, runbook automation
- **Security professionals** — audit, compliance, threat analysis

### Secondary: Knowledge Workers

- **Writers** — research, drafting, editing workflows
- **Analysts** — data analysis, report generation
- **Consultants** — client deliverables, competitive analysis

## Revenue Model

### Open Core

| Tier | Contents | Price |
|------|----------|-------|
| **Community** | CLI + TUI + Desktop, all core crates, 22 skills | Free (MIT) |
| **Pro** (future) | Engine server, team sessions, mission orchestration | Subscription |
| **Enterprise** (future) | Multi-instance federation, SSO, priority support | Custom |

### Revenue Streams

1. **Pro subscriptions** — hosted Engine for non-technical users
2. **Marketplace** — premium skill packs, integrations
3. **Support contracts** — enterprise deployment assistance
4. **Training** — workshops on building AI agent workflows

## Growth Strategy

### Phase 1: Developer Adoption (Current)

- Open-source core and agent on GitHub
- Developer blog posts and tutorials
- Reddit, Hacker News, and dev community engagement
- CLI-first UX that developers love

### Phase 2: Community Building

- Discord community
- Skill marketplace (user-contributed skills)
- MCP server directory
- Plugin ecosystem

### Phase 3: Commercial Launch

- Pro tier with hosted Engine
- Desktop app in app stores
- Marketing website at aivyx.ai
- Content marketing: privacy-focused AI positioning
