## About Aivyx

**Aivyx** is an open-source, privacy-first AI agent that runs locally on your machine. It encrypts all data at rest, supports multiple LLM providers, and includes 22 built-in professional skills.

- **Website**: [aivyx.ai](https://aivyx.ai)
- **GitHub**: [github.com/AivyxDev/aivyx](https://github.com/AivyxDev/aivyx)
- **License**: MIT (agent + core), Proprietary (engine)
- **Language**: Rust
- **Platforms**: Linux, macOS, Windows

## Boilerplate

### Short (25 words)

### Medium (50 words)
> Aivyx is an open-source AI agent built for professionals who care about privacy. It runs locally, encrypts all data with ChaCha20Poly1305, supports multiple LLM providers including local models via Ollama, and includes 22 built-in skills for development, research, and analysis workflows.

### Long (100 words)
> Aivyx is an open-source, privacy-first AI agent designed for developers, researchers, and knowledge workers. Built in Rust, it runs entirely on your machine with no cloud dependency — supporting local models via Ollama or external providers like OpenAI, Anthropic, and Google Gemini. All data is encrypted at rest using ChaCha20Poly1305 with HKDF-derived keys, and every action is recorded in a tamper-proof HMAC audit chain. With 22 built-in professional skills, Model Context Protocol tool connectivity, a full terminal UI, and a Tauri desktop app, Aivyx gives professionals AI capabilities without sacrificing data ownership.
